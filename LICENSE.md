On Windows systems, configurations are stored locally in the `%appdata%/MM2Utility/config.json` directory. On macOS systems, the settings file is located at `~/Library/Application Support/MM2Utility/config.json`.

| Variable | Default Value | Description |
| :--- | :--- | :--- |
| `SilentAimFOV` | `120.0` | The radius for the automated aiming field of view (in pixels). |
| `SafeMode` | `true` | Enforces randomized human-like delays for key actions. |
| `ESPSelection` | `"Both"` | Visual tracking settings (Murderer, Sheriff, or Both). |
| `CandyAutoFarm` | `false` | Enables/disables automatic event currency collection loops. |
